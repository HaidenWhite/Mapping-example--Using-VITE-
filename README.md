This program isn't designed to be fully stand-alone. 
It will compile, run and be accessed through a browser though. 
But it serves as a useable base or reference for any further typescript project that requires a map item.

Feel free to use!

Requirements:
- Node.js command prompt
- Microsoft visual studio code
- Leaflet and it's types
- typescript tsx
- express and it's types

Optional requirements:
- If you wish to access markers through a database, better-sqlite-3 is required
- pm2 to keep the server running indefintely
